# 加入 Vectie：Agents、LLM 与 Robotics

欢迎来到 Vectie。

Vectie 是一个关注 agents、LLM 和 robotics 的开源组织。如果你对这些方向感兴趣，欢迎在这个讨论串里做个自我介绍。

## 可以介绍什么

- 你的 GitHub 主页
- 你当前关注的方向：agents、LLM、robotics、tooling、infrastructure 或 research
- 你想参与的项目
- 你做过的相关项目、文章或作品链接
- 你希望参与的方式：代码、文档、实验、测试、设计或运维

## 参与路径

- 先从 fork 项目、提交 PR 开始
- 尽量保持提交范围小、描述清晰、方便 review
- 持续稳定的高质量贡献会带来更深入的协作机会
