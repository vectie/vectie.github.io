# Join Vectie: Agents, LLMs, and Robotics

Welcome to Vectie.

Vectie is an open-source group for people who want to build practical systems around agents, LLMs, and robotics. If you are interested in these areas, introduce yourself in this thread.

## What To Share

- Your GitHub profile
- Your current interests: agents, LLMs, robotics, tooling, infrastructure, or research
- Projects you want to work on
- Links to anything relevant you have built or written
- How you would like to contribute: code, docs, experiments, testing, design, or operations

## Contribution Path

- Start by forking a project and opening pull requests
- Keep contributions small, clear, and reviewable
- High-quality sustained contributions can lead to broader collaboration
